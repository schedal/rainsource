This project is for http://rainsource.net a permaculture website built in http://Drupal.org. This website's first release will use AJAX to load various pieces of information via a custom module to the petals of each service sector they offer.

Fully customized premium design, project also uses LESS for CSS compiling and a minimized web-admin role / theme.
